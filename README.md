# Project Management Portal

## Overview
Project Management Portal is a web-based application developed to simplify project planning, task management, team collaboration, and project tracking. The platform enables administrators, project managers, and team members to efficiently manage projects, assign tasks, monitor progress, and generate reports through a centralized system.

## Features

### User Management
- User Registration and Authentication
- Secure Login and Logout
- Role-Based Access Control (Admin, Manager, Team Member)

### Project Management
- Create, Update, and Delete Projects
- Define Project Deadlines and Milestones
- Assign Team Members to Projects
- Monitor Project Status

### Task Management
- Create and Assign Tasks
- Set Priorities and Deadlines
- Update Task Progress
- Track Task Completion Status

### Dashboard
- Project Overview
- Active Projects Count
- Completed Projects Count
- Pending Tasks Summary
- Team Performance Statistics

### Reports
- Project Progress Reports
- Task Completion Reports
- User Activity Reports
